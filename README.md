# penguinpay-prototype
### _Built on Android Studio 4.2.2_

Before running the project, add the apiKey string property (from: https://openexchangerates.org) to your `local.properties` file.

## Stack

- MVVM
- Clean Architecture
- Coroutine
- Retrofit + Moshi
- Koin
- Lottie
